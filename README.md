# 🎤 Oratory Bot — Бот по ораторскому искусству

Полноценный Telegram-бот с Claude AI, логированием, уведомлениями и геймификацией.

---

## 📁 Структура проекта

```
oratory_bot/
├── bot.py           ← главный файл (запускать его)
├── config.py        ← твои токены и настройки
├── database.py      ← работа с базой данных
├── prompts.py       ← промпт, фильтры, уровни, задания
├── requirements.txt ← зависимости
├── README.md        ← эта инструкция
└── data/
    └── bot_logs.db  ← создаётся автоматически при запуске
```

## ⚙️ Шаг 1 — Получи токены

### Токен бота (Telegram)
1. Открой Telegram → найди @BotFather
2. Напиши `/newbot`
3. Придумай имя и username боту
4. Скопируй токен вида: `7123456789:AAHx...`

### Твой Telegram ID (ADMIN_CHAT_ID)
1. Открой @userinfobot в Telegram
2. Напиши `/start`
3. Скопируй число в поле "Id:" — это твой ID

### API ключ Claude (Anthropic)
1. Зайди на https://console.anthropic.com
2. Зарегистрируйся / войди
3. Перейди в раздел "API Keys"
4. Нажми "Create Key" → скопируй ключ вида: `sk-ant-...`

---

## ⚙️ Шаг 2 — Заполни config.py

Открой файл `config.py` и замени значения:

```python
BOT_TOKEN = "7123456789:AAHx..."        # ← твой токен от BotFather
ADMIN_CHAT_ID = "123456789"             # ← твой Telegram ID
ANTHROPIC_API_KEY = "sk-ant-..."        # ← ключ Claude
```

## 🖥 Шаг 3 — Запуск на компьютере (для теста)

### Требования: Python 3.10+

```bash
# 1. Установи зависимости
pip install -r requirements.txt

# 2. Запусти бота
python bot.py
```

Бот запустится. Открой Telegram → найди своего бота → напиши /start

## 🌐 Шаг 4 — Запуск на сервере (постоянная работа 24/7)

### Вариант А: Railway.app (бесплатно, просто)
1. Зайди на https://railway.app
2. Создай новый проект → "Deploy from GitHub"
3. Загрузи папку бота на GitHub
4. В настройках Railway добавь переменные окружения:
   - `BOT_TOKEN` = твой токен
   - `ADMIN_CHAT_ID` = твой ID
   - `ANTHROPIC_API_KEY` = ключ Claude
5. Нажми Deploy — всё!

### Вариант Б: VPS сервер (Beget, Timeweb, Hetzner)
```bash
# Подключись по SSH, затем:
git clone <твой репозиторий>
cd oratory_bot
pip install -r requirements.txt

# Запуск в фоне (не останавливается при закрытии терминала)
nohup python bot.py &

# Или через systemd (рекомендуется для продакшена)
```

### Вариант В: Render.com (бесплатно)
1. Зайди на https://render.com
2. New → Web Service → подключи GitHub
3. Build Command: `pip install -r requirements.txt`
4. Start Command: `python bot.py`
5. Добавь Environment Variables (те же 3 ключа)

## 🗃 Где хранятся данные

Все данные хранятся в файле `data/bot_logs.db` (SQLite).

**Таблица `logs`** — каждый вопрос и ответ:
- timestamp — дата и время
- user_id — ID пользователя
- username — @username
- full_name — имя
- question — вопрос
- bot_answer — ответ бота
- was_blocked — 1 если заблокировано

**Таблица `users`** — каждый пользователь:
- message_count — сколько вопросов задал
- level — уровень (Новичок → Спикер → Оратор → Мастер → Легенда)
- first_seen — когда первый раз написал

### Как посмотреть базу данных
Скачай бесплатный инструмент: https://sqlitebrowser.org
Открой файл `data/bot_logs.db` — увидишь все данные в виде таблицы.

## ❓ Часто задаваемые вопросы

**Бот не отвечает — что делать?**
- Проверь правильность токенов в config.py
- Убедись что бот запущен (видишь "🚀 Бот запущен!" в терминале)

**Не получаю уведомления**
- Проверь ADMIN_CHAT_ID — должно быть число, не username
- Сначала напиши боту /start чтобы "открыть" чат

**Как изменить темы или добавить слова в фильтр?**
- Открой `prompts.py` → добавь слова в список BLOCKED_KEYWORDS
